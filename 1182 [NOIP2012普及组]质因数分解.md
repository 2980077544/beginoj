# 1182 [NOIP2012普及组]质因数分解
#### Description
已知正整数n是两个不同的质数的乘积，试求出较大的那个质数。
#### Input
只有一行，包含一个正整数n。
6 ≤ n ≤ 2*10^9
#### Output
输出只有一行，包含一个正整数p，即较大的那个质数。
#### Sample Input
```
21
```
#### Sample Output
```
7
```
#### HINT
* * *
```
#include <bits/stdc++.h>
using namespace std;
int main()
{
    int a,t=0,n=10;
    cin>>a;
    for(int i=10;i<=n;i*=10)
    {
        if(a%n/(n/10)==8) t++;
        if(a/i>1) n*=10;
    }
    cout<<t<<endl;
    return 0;
}
```
